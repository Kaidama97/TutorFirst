TutorFirst

Proposed Level of Achievement (Apollo 11)

TutorFirst is dedicated to enhancing the educational experience by connecting students with qualified tutors through an intuitive and user-friendly platform. Our goal is to make finding and booking tutoring sessions seamless and effective for both students and educators.

 

Project Scope

One Sentence Version
TutorFirst aims to develop an intuitive online platform that connects students with qualified tutors, streamlining the process of finding, scheduling, and managing tutoring sessions.

Longer Descriptive Version

TutorFirst is an innovative online platform designed to bridge the gap between students seeking academic support and qualified tutors offering their expertise. The project scope encompasses the development of a user-centric platform that facilitates seamless interactions between students and tutors. Key features include a sophisticated search function to help students find tutors based on subject expertise and availability, a robust booking system with real-time scheduling capabilities, and detailed tutor profiles with qualifications and subject area. The platform will also integrate secure messaging features to enable effective communication between users, and a recommendation system that provides personalized tutor suggestions based on user preferences and history. The project aims to ensure mobile compatibility, allowing users to manage their tutoring needs from any device. By focusing on ease of use, efficiency, and personalization, TutorFirst seeks to enhance the educational experience and streamline the tutoring process for both students and educators.

Project Setup

*Prerequisites*

- Git: Ensure Git is installed on your machine. You can download and install it from git-scm.com.

- Node.js: Make sure you have Node.js installed. You can download it from Node.js official website.

- React Native CLI: Install the React Native CLI globally: npm install -g react-native-cli

- Expo CLI: Install the Expo CLI globally using npm or yarn: npm install -g expo-cli
